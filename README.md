    - 1️⃣ What is the difference between var, let, and const?
          var => old, function scope, can be redeclared.
          let => block scope, value can be changed.
          const => block scope, value cannot be changed.

    - 2️⃣ What is the spread operator (...)?
    ... can be used to copy or merge the elements of an array or object separately.

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
    map() => returns a new array.
    filter() => returns a new array according to the condition.
    forEach() => just loops, does not return anything.

    - 4️⃣ What is an arrow function?
    ES6 syntax for writing short and simple functions.

    - 5️⃣ What are template literals?
    Backtick ` ` can be used to write strings and ${} can be used to set variables.
